# Telegram 双向机器人（Cloudflare D1 + Workers）

基于 Cloudflare D1 和 Workers 的 Telegram 双向机器人，支持用户私聊消息转发到群组话题，管理员在话题中回复自动转发回用户。内置高性能屏蔽词系统（Aho-Corasick 自动机 + 正则）、多实例缓存、消息去重、日志分级等特性。

## ✨ 功能特性

- **双向转发**：用户私聊机器人 → 群组话题；管理员话题回复 → 用户
- **多用户隔离**：每个用户在群组中拥有独立话题
- **高性能屏蔽词**：
  - 字面量预处理（忽略标点空格，保留字母/数字/Emoji）使用 AC 自动机
  - 字面量精确匹配（原始包含）
  - 正则匹配（长度/数量限制，慢查询监控）
- **多实例缓存一致性**：内存缓存 + KV 快照 + 版本号控制
- **消息去重**：内存 LRU 缓存 + KV 全局去重，防刷
- **管理员命令**：拉黑/解封、增删屏蔽词、分页列表、清理旧话题
- **日志分级与采样**：支持 DEBUG/INFO/WARN/ERROR，DEBUG 日志可采样
- **监控统计**：拦截记录、命令日志、正则性能统计接口 `/stats`
- **API 重试**：自动重试 429 和 5xx 错误，指数退避
